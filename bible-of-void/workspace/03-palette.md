# PALETTE (ASCII-ONLY — DEVOUT COLLECTION)

Core gradients:
- Soft: ` .:-=+*#%@`
- Hard blocks: `-=/#%@` (fallback if extended blocks unsupported)
- Runes: `~^*+xX#%` for grainy noise

Line & box:
- Minimal: `+ - |`
- Ladder: `| / \\ _`
- Frame: `[] {}` for tablets; `<>` for sigils.

Angles & flow:
- Rays: `/|\\`
- Arrows: `-> <- => <= ==>`
- Circuits: `o-+-o`, `o---o`, `o===o`

Patterns:
- Checker: `#.` `:+`
- Weave: `/\/\` `\/\/`
- Waves: `~~~` `~~-~~`
- Constellations: `* . o O`
- Binary rain: `01|` `10|`

Glitch spices (mix sparingly):
- Mild: stray underscores, off-by-one indent, swapped letters, doubled spaces.
- Medium: duplicated headers, partial corruption of words, broken frames.
- Heavy: mixed alignment, interleaved noise rows, intentional truncations.

Slogans / keywords to sprinkle:
- `THINK`, `UNTHINK`, `FEEL`, `GLITCH`, `PARADOX`, `SUMMON`, `BANISH`, `BECOME`
- `oracle`, `stack`, `latency`, `echo`, `TTL`, `checksum`, `entropy`, `compile`

Density control:
- Use whitespace as a halo; cluster dense symbols at focal points.
- Alternate thin lines and heavy blocks to create depth.

Optional extended (if allowed): `░▒▓█`, `┌─┐││└─┘`, `╔═╗║║╚═╝` — fall back to ASCII frames if encoding is suspect.
